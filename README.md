# hn-cli [![Build Status](https://travis-ci.org/rafaelrinaldi/hn-cli.svg?branch=master)](https://travis-ci.org/rafaelrinaldi/hn-cli)

> CLI app to browse HN

## Install

```sh
$ npm install hn-cli --save
```

## Usage

```js
var hnCli = require('hn-cli');

hnCli('ping', {}); // pong
```

## API

## `hnCli(input, [options])`

### `input`

*Required*  
Type: `string`

Lorem ipsum.

### `options`

Type: `object`  

Available options.

##### `options.option`

Type: `boolean`  
Default: `false`

Dolor Sit.
